# Zytra Analytics

Zytra Analytics is a web-based demand forecasting and business intelligence platform
designed to help small and mid-sized businesses make data-driven decisions.

## Features
- Secure user authentication
- Multi-file CSV data upload
- Interactive visual analytics
- Demand forecasting using SARIMA and Prophet
- Business impact insights
- Downloadable reports

## Tech Stack
- Python
- Streamlit
- Pandas
- Statsmodels
- Prophet

## Use Case
Retail demand forecasting, inventory planning, and trend analysis.
